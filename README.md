# RaceWayKart Application

<img src="./readme-resources/logo.png" alt="logo_race_way_kart" width="150"/>

Yo hey! This is an application for creating groups of racing cart races for track RaceWayKart.

# How to use it

1. Install IntelliJ IDEA if you don't have the program yet

2. Open IntelliJ IDEA: File -> New -> Project from Version Control...
<div><img src="./readme-resources/new_project_from_version_control.png" width= "450"/></div>

3. Enter a link to the repository in the URL field and click Clone
```
URL: https://github.com/tquicklyy/race_way_kart_app
```
<div> <img src="readme-resources/enter_link_from_version_control.png" alt="Image 2" width="450" /></div>

4. Run and use the program.

# General information

The software provides two use cases depending on the stage of the race: Race and Qualification.
When the stage changes, the logic and appearance of the app changes.

## Qualification stage
<div> <img src="readme-resources/qual_stage_select.png" alt="Image 2" width="200" /></div>

By selecting the qualification stage as shown in the image to get a list of groups (drivers + their karts), you need to:
1) Enter driver info in the drivers window;
2) Enter kart info in the karts window;
3) Enter the number of groups and start generation in the groups window.

## Race stage
<div> <img src="readme-resources/race_stage_select.png" alt="Image 2" width="200" /></div>

The qualifying results divide the drivers into groups, but it necessary to issue carts, so by selecting the race mode as shown in the image, you can get a list of groups as follows:
1) Enter driver info along with the group number in the drivers window;
2) Enter kart info in the karts window;
3) Start generation in the groups window

## Additional features
The application can also:
- Save data between launches;
- Reset all existing data;
- Reset individual data in the appropriate windows;
- Save information to a .txt file in a special readable format.
